# Nomu-Sup（ノムサプ）｜ 薬剤師監修・二日酔い対策ソムリエ

## サービス概要
「飲み会前」「飲酒中」「翌朝」の3つのタイミングに合わせて、**薬学的根拠に基づいた「今、コンビニや薬局で買うべき対策セット」を提案**するWebアプリです。
ユーザーの状態を選択するだけで、薬剤師の思考プロセスを再現したアルゴリズムが、最適なサプリメント・市販薬・食品の組み合わせをレコメンドします。

---

## このサービスへの思い・作りたい理由
薬剤師として働く中で、二日酔いに苦しむ人々が「脱水状態なのにカフェインを摂る」といった、医学的に逆効果な対処法を選んでしまっている場面を多く見てきました。
「科学的に正しいタイミングで、正しい成分を摂取すれば、翌日のパフォーマンスは守れる」という確信があります。
しかし、飲み会の最中や二日酔いの時に専門情報を調べるのは困難です。そこで、**「酔っ払っていても3タップで専門家の正解に辿り着けるツール」**を提供し、人々の健康と翌日の生産性を守りたいと考え開発しました。

## ユーザー層について
* **ターゲット：** 20〜30代の働く世代・学生
* **選定理由：** * 付き合いでの飲酒機会が多く、翌日の仕事や学業への影響を最小限にしたいという切実なニーズがある。
    * 健康管理のために「コンビニで数百円の対策」を講じるハードルが低く、スマホでの情報検索が習慣化しているため。

## サービスの利用イメージ
1.  **飲み会直前（予防）：** 「これから飲む」を選択。空腹状態や飲む量に応じ、胃粘膜保護（乳製品）や肝機能補助（ヘパリーゼ等）の最適なセットが提案されます。
2.  **飲み会後（代謝促進）：** 「飲みすぎた」を選択。アルコール分解を助ける糖分（ラムネ）や水分、代謝を早める特定食品を提案し、翌朝へのダメージを軽減します。
3.  **翌朝（対症療法）：** 「頭痛」「吐き気」などの症状を選択。薬剤師のロジックに基づき、胃への負担を考慮した鎮痛剤（アセトアミノフェン等）や、漢方薬（五苓散等）の具体的な商品名を提案します。

## ユーザーの獲得について
* **職場・知人へのテスト導入:** まずは職場の同僚や友人にプロトタイプを共有し、実際の飲み会前後に使用してもらうことで、実用性に関するフィードバックを直接収集します。
* **対面での紹介:** 薬剤師としての繋がりや、知人のコミュニティにて「二日酔い対策のガイド」として紹介し、ユーザーの生の声を元にUI/UXの改善を繰り返します。

## サービスの差別化ポイント・推しポイント
* **明確な解決策の提示：** 単なる記録や注意喚起ではなく、「何を買えばいいか」という具体的な商品・成分を提示します。
* **薬剤師ロジックの搭載（推しポイント）：** * 「胃が痛いならこの鎮痛剤は避ける」といった、専門家ならではの安全性を考慮した分岐ロジック。
    * 生薬（漢方）、アミノ酸、ビタミン、食品の相互作用を考慮した多角的なレコメンド。

## 機能候補

### MVP
* **3ステップ診断機能：** タイミング × 状態 × 症状 によるレコメンド出力。
* **対策カード表示：** 推奨成分と、コンビニ等で購入可能な代表的商品名の表示。
* **ワンポイント解説：** 「なぜこの成分が必要か」の薬学的根拠（EBM）を表示。

### 本リリース
* **アルコール分解シミュレーター：** 体重・飲酒量から「酔いが覚める目安時間」を算出。
* **マイ薬箱ログ：** 過去に効果があった対策を保存し、自分専用の対策集を作成。
* **店舗検索API連携：** 現在地近くの24時間営業ドラッグストアを地図上に表示。
